# Demo Admin Frontend

React + Vite + TypeScript frontend for the admin console (login, user list, create/edit).

## Prerequisites
- Node.js 18+

## Running
```bash
cd frontend
npm install
npm run dev
```

The dev server runs on `http://localhost:5173` and proxies API calls to `http://localhost:8080/api` (backend must be running).

## Available scripts
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build locally
